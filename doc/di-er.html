6.1 Donut

23 Haziran 2019 İstanbul sonuçları:

#En basit şekilde

g5_1$Oran <- g5_1$Oy / sum(g5_1$Oy) #Oy oranları
g5_1$ymax <- cumsum(g5_1$Oran) #Kümülatif oranlar
g5_1$ymin = c(0, head(g5_1$ymax, n = -1)) #Alan

ggplot(g5_1, aes(ymax = ymax, ymin = ymin, xmax = 4, xmin = 3, fill = Aday)) +
  geom_rect() +
  coord_polar(theta = "y") +
  xlim(c(2, 4))

  • İsimlerin pozisyonlarını ayarlayabiliriz.

  • Etiketlerin gösterim şeklini ayarlayabiliriz.

  • Renklendirme yapabiliriz.

g5_1$EtiketPoz <- (g5_1$ymax + g5_1$ymin) / 2
g5_1$Etiket <- paste0(g5_1$Aday, "\n%", round(g5_1$Oran * 100, digits = 2))

ggplot(g5_1, aes(ymax = ymax, ymin = ymin, xmax = 4, xmin = 3, fill = Aday)) +
  geom_rect() +
  geom_text(x = 3.5, aes(y = EtiketPoz, label = Etiket), size = 3) +
  coord_polar(theta = "y") +
  xlim(c(2, 4)) +
  scale_fill_manual(values = c("orange", "gray", "red"))

  • Lejantı kaldırabiliriz.

  • Başlık, alt başlık ve kaynak ekleyebiliriz.

  • Temayı değiştirebiliriz.

ggplot(g5_1, aes(ymax = ymax, ymin = ymin, xmax = 4, xmin = 3, fill = Aday)) +
  geom_rect() +
  geom_text(x = 3.5, aes(y = EtiketPoz, label = Etiket), size = 3) +
  theme_void() +
  coord_polar(theta = "y") +
  xlim(c(2, 4)) +
  scale_fill_manual(values = c("orange", "gray", "red")) +
  theme(legend.position = "none") +
  labs(title = "İstanbul Seçim Sonuçları",
       subtitle = "23 Haziran 2019",
       caption = "Kaynak: NTV")
