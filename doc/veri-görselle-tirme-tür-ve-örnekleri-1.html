4.1.1 Histogram ve Yoğunluk (Histogram-Density)

Son 10 yıla ait günlük BIST Endeksi toplam işlem hacmi:

#En basit şekilde

ggplot(data = v3) +
  geom_histogram(mapping = aes(x = `ToplamİşlemHacmi`))

  • Aralık seçimini ayarlayabiliriz (Ör: 500000).

  • x eksenine ait değerleri daha okunabilir bir formata getirebiliriz.

  • Renklendirme yapabiliriz.

ggplot(data = v3) +
  geom_histogram(mapping = aes(x = `ToplamİşlemHacmi`), binwidth = 500000, fill = "dark red") +
  scale_x_continuous(labels = scales::comma)

  • x ve y eksenlerine ait başlığı düzeltebiliriz.

  • Ortalama çizgisi (kesikli) ekleyebiliriz.

  • Başlık, alt başlık ve kaynak ekleyebiliriz.

  • Temayı değiştirebiliriz.

ggplot(data = v3) +
  geom_histogram(mapping = aes(x = `ToplamİşlemHacmi`), binwidth = 500000, fill = "dark red") +
  geom_vline(xintercept = mean(v3$`ToplamİşlemHacmi`), linetype = "dashed") +
  scale_x_continuous(labels = scales::comma) +
  theme_minimal() +
  labs(x = "Toplam İşlem Hacmi",
       y = "Sayma (Count)",
       title = "Günlük BIST Endeksi Toplam İşlem Hacmi",
       subtitle = "Son 10 yıla ait verilerdir",
       caption = "Kaynak: TCMB")

  • Sayma (count) yerine yoğunluk (density) olarak göstermek için;
#Bazı yerleri değiştirdik; aradaki farklara dikkat edin.

ggplot(data = v3, mapping = aes(x = `ToplamİşlemHacmi`)) +
  geom_histogram(mapping = aes(y = ..density..), binwidth = 500000, fill = "dark red", alpha = 0.5) +
  geom_density(color = "blue") +
  geom_vline(xintercept = mean(v3$`ToplamİşlemHacmi`), linetype = "dashed") +
  scale_x_continuous(labels = scales::comma) +
  scale_y_continuous(labels = scales::comma) +
  theme_minimal() +
  labs(x = "Toplam İşlem Hacmi",
       y = "Yoğunluk (Density)",
       title = "Günlük BIST Endeksi Toplam İşlem Hacmi",
       subtitle = "Son 10 yıla ait verilerdir",
       caption = "Kaynak: TCMB")

4.1.2 Kutu-Bıyık (Boxplot-Whisker)

Son 1 yıla ait günlük BIST Hizmet, Mali, Sınai, Teknoloji endeks kapanışları:

#En basit şekilde

ggplot(data = v4) +
  geom_boxplot(mapping = aes(x = Endeks, y = `KapanışFiyatları`))

  • y eksenine ait değerleri daha okunabilir bir formata getirebiliriz.

  • Kutuları renklendirip palet seçenekleri ile renkleri değiştirebiliriz.

ggplot(data = v4) +
  geom_boxplot(mapping = aes(x = Endeks, y = `KapanışFiyatları`, fill = Endeks)) +
  scale_y_continuous(labels = scales::comma) +
  scale_fill_brewer(palette = "Set1")

  • Lejantı kaldırabiliriz.

  • x eksenine ait başlığı kaldırabilir; y eksenine ait başlığı düzeltebiliriz.

  • Grafiğe başlık, alt başlık ve kaynak ekleyebiliriz.

  • Temayı değiştirebiliriz.

ggplot(data = v4) +
  geom_boxplot(mapping = aes(x = Endeks, y = `KapanışFiyatları`, fill = Endeks)) +
  theme_minimal() +
  theme(legend.position = "none") +
  scale_y_continuous(labels = scales::comma) +
  labs(x = NULL,
       y = "Endeks Kapanışları",
       title = "Günlük BIST Hizmet, Mali, Sınai, Teknoloji Endeks Kapanışları",
       subtitle = "Son 1 yıla ait verilerdir",
       caption = "Kaynak: TCMB") +
  scale_fill_brewer(palette = "Set1")
